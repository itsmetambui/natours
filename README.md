# Natours
A simple project to showcase frontend skills (HTML/CSS)

## Getting Started

```
yarn install
yarn start
```

## Deployment

This repo contains now.sh to deploy to [NOW](https://zeit.co/now), simply run

```
yarn build:css
now
```
